# Patch Embedded k-NN Graph Neural Networks for Wafer Defect Classification


## 서론

### 1. CNN 기반 연구의 한계
기존 CNN 모델은 합성곱 필터를 사용하여 이미지의 모양(현재 논문에서는 웨이퍼의 결함을 의미)를 학습한다. 이러한 합성곱 필터는 웨이퍼 전체의 좁은 영역(국소적 특징)만 볼 수 있기에 결함의 전체 형태를 완전히 학습하기 어렵다. 또한 합성곱 필터를 통해 입력된 input 정보들을 여러 신경망 계층을 여러 번 거쳐서 추측하기 때문에 Scratch 결함과 불량 Die 간 거리가 긴 결함의 경우 해당 결함을 파악하기 어렵다. 마지막으로 CNN은 불필요한 배경 픽셀까지 훑기 때문에 결함의 구조를 직관적으로 파악하는 것이 어렵다.

### 2. CNN + GCN
CNN Incoder에서 추출된 특징 임베딩이 Graph의 노드로 해석되며 각 특징들의 노드들이 연결되어 노드 간의 관계가 명시적으로 정의된다. 이 노드 간의 관걔를 취합하는 전략이 있다.

GCN: 주변에 연결된 이웃 노드들의 정보를 모두 더한 뒤, 이웃의 수만큼 공평하게 '평균(N분의 1)'을 내어 받아들입니다.

GraphSAGE: 이웃 노드가 너무 많으면 계산이 느려진다. 그래서 모든 이웃을 보지 않고 무작위로 일부만 샘플링하여 정보를 취합한다. 학습 속도가 빠르며 새로운 데이터에도 유연하게 대처한다.

GAT: 현재 결함을 파악하는데 결정적인 정보를 주는 핵심 노드만 높은 가중치를 주어 정보를 흡수한다. 형태가 불규칙한 경우에 가장 정확하게 작동하는 방식이다.

### 3. 산업적 응용 분야에서의 GCN 적용
철강 표면 결함 분류를 위한 GNN의 효과가 입증되었으며, 이는 그래프 기반 특징 전파를 통해 국부적 상호작용과 전역적 구조적 종속성을 모두 포착하는 능력을 보여주었다. 유사하게 항공기 엔진 블레이드 결함 분류를 위해 슈퍼픽셀 인식 그래프 신경망(SPGNN)이 도입되기도 했다.

### 4. 목적
본 연구는 CNN 인코더에서 추출된 특징 표현을 기반으로 백본 역할을 하는 GAT를 통합하는 단일 아키텍처를 개발하였다. 이를 위해 GCN, GraphSAGE 등의 모델도 비교군으로 포함하여 평가를 진행하였다. 또한 데이터셋을 30번을 다르게 쪼개서 테스트하여 Robustness와 일반화 능력을 입증하고자 한다.


## 본론

### 1단계: 데이터 전처리 및 증강 (Data Preprocessing & Augmentation)
심각한 클래스 불균형 문제를 해결하고 다양한 크기의 웨이퍼를 모델에 입력하기 위해 데이터를 표준화하는 단계입니다.

크기 표준화: 형태 왜곡을 막기 위해 '최근접 이웃 보간법(nearest-neighbor interpolation)'을 사용하여 모든 웨이퍼 빈 맵을 128×128 크기로 고정합니다.

데이터 증강: 극심한 클래스 불균형을 해소하기 위해 훈련 세트의 소수 결함 클래스들에 기하학적 변환(무작위 뒤집기, 회전, 스케일링 등)을 적용하여 각 클래스의 샘플 수를 6,970개로 증식합니다.

원-핫 인코딩: 배경, 정상 다이, 결함 다이를 구분하기 위해 3채널 이미지 형태로 변환합니다.

### 2단계: 패치 임베딩 (Patch Embedding - 그래프 노드 생성)
웨이퍼 맵을 이미지 전체로 보지 않고 여러 조각(Patch)으로 나누어, 각각을 그래프의 '노드(Node)'로 만드는 단계입니다.

CNN 기반 Patchifier: 3채널의 웨이퍼 이미지를 CNN에 통과시켜 64개(8×8 해상도)의 패치로 분할합니다.

특징 벡터 변환: 분할된 각 패치는 256차원의 임베딩 벡터로 변환됩니다.

위치 임베딩 추가: 패치로 나누는 과정에서 손실될 수 있는 공간적 위치 정보를 보존하기 위해, 학습 가능한 위치 임베딩(positional embedding)을 각 패치 특징에 더해줍니다.

### 3단계: 동적 k-NN 그래프 구성 (k-NN Graph Construction - 노드 간 연결)
추출된 64개의 패치들을 서로 연결하여 하나의 그래프 구조로 만드는 단계입니다.

유클리드 거리 기반 연결: 고정된 픽셀 위치가 아니라, 임베딩 공간 내에서 특징 벡터 간의 유클리드 거리를 계산하여 의미적으로 유사한 패치들을 서로 연결합니다.

동적 k-NN 적용: 각 패치(노드)를 기준으로 가장 거리가 가까운 k개의 노드(실험에서는 k=7로 설정)를 이웃으로 정의하여 방향성 엣지(edge)를 형성합니다. 

이를 통해 멀리 떨어져 있는 결함 영역 간의 관계도 파악할 수 있습니다.

### 4단계: 특징 집계 및 최종 분류 (GAT & Classification)
완성된 그래프를 바탕으로 패치 간의 정보를 교환하고 최종 결함 유형을 판별하는 단계입니다.

다중 헤드 GAT (Graph Attention Network): 어텐션 메커니즘을 사용해 이웃 노드들의 정보를 수집합니다. 이때 모든 이웃의 정보를 똑같이 반영하는 것이 아니라, 학습된 어텐션 계수에 따라 가중치를 다르게 주어 중요한 결함 정보에 집중합니다.

잔차 연결 및 판독(Readout): GNN이 깊어질수록 특징이 뭉개지는 현상(oversmoothing)을 막기 위해 잔차 연결(residual connection)을 사용하며, 최종적으로 적응형 평균 풀링을 거쳐 512차원의 단일 웨이퍼 수준 벡터를 생성합니다.

최종 분류: 2개의 은닉층을 가진 완전 연결 신경망(FCNN)과 소프트맥스(softmax) 함수를 거쳐, 9가지의 웨이퍼 결함 클래스 중 하나로 최종 분류(Cross-entropy loss 최소화)합니다.
